# GameStore - Guide d'Utilisation

## Bienvenue
GameStore est votre application de gestion de catalogue de jeux vidéo. Elle vous permet de consulter la liste des jeux disponibles, d'en ajouter de nouveaux, et de gérer les informations existantes.

---

## 1. Accès au site
En tant que visiteur simple, vous pouvez accéder à la page d'accueil et consulter la liste des jeux.
Cependant, pour modifier le catalogue, vous devez vous connecter en tant qu'administrateur.

## 2. Connexion (Espace Admin)
Certaines fonctionnalités (Ajouter, Modifier, Supprimer) sont protégées.
1. Cliquez sur le bouton **"Connexion"** ou **"Espace Admin"** dans le menu en haut à droite.
2. Entrez vos identifiants :
   * **Email :** `admin@ecole.fr`
   * **Mot de passe :** `admin123`
3. Une fois connecté, votre prénom s'affiche dans la barre de menu et les boutons d'administration apparaissent.

## 3. Gérer les Jeux

### ➤ Ajouter un jeu
1. Assurez-vous d'être connecté.
2. Sur la page "Catalogue Jeux", cliquez sur le bouton bleu **"Nouveau Jeu"**.
3. Remplissez le formulaire (Titre, Description, Date, Éditeur, PEGI).
4. Cliquez sur **"Enregistrer"**. Le jeu apparaît immédiatement dans la liste.

### ➤ Modifier un jeu
1. Repérez le jeu à modifier dans la liste.
2. Cliquez sur le petit bouton jaune avec un crayon **(✏️)**.
3. Modifiez les informations souhaitées.
4. Validez pour sauvegarder.

### ➤ Supprimer un jeu
1. Repérez le jeu à supprimer.
2. Cliquez sur le bouton rouge avec une corbeille **(🗑️)**.
3. Confirmez la suppression dans la fenêtre qui s'ouvre. **Attention, cette action est irréversible.**

## 4. Gestion des Genres
Vous pouvez également gérer les catégories de jeux via le menu **"Liste des Genres"**. Le fonctionnement est identique (Ajout, Modification, Suppression).
